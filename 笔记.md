## 循环
使用嵌套循环输出乘法表
````java
    for(int i=1;i<=9;i++){
        for( int m=1;m<=i;m++){
                System.out.print(i+"*"+m+"="+(m*i)+"\t");
        }
        System.out.println();
        }
````

## 构造方法与类
要点：
1. 构造器通过new关键字调用！！
2. 构造器虽然有返回值，但是不能定义返回值类型（返回值的类型肯定是本类），不能在构造器里使用return返回某个值。
3. 如果我们没有定义构造器，则编译器会自动定义一个无参的构造方法。如果已定义则编译器不会自动添加！
4. 构造器的方法名必须和类名一致。
